# CountDownView

+![TimePicker](https://github.com/chenxiruanhai/CountDownView/blob/master/CountDownView.jpg)

#倒计时组件     支持 XX分XX秒 格式的倒计时。

| attr 属性          | description 描述 |
|:---				 |:---|
| countDownTime      | 倒计时时间 |
| count_down_format | 倒计时样式本|

#Method
###1.  setCountDownTimes()
 设置倒计时时间</br> 
###1.  start()
 启动倒计时</br> 
###1.  setOnCountDownFinishListener()
 倒计时结束监听</br>
