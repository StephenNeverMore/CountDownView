#倒计时组件     支持 XX分XX秒 格式的倒计时。

   支持 XX分XX秒 格式的倒计时
   attrs:
         countDownTime 倒计时时间
	 count_down_format 倒计时样式本
   method:
         setCountDownTimes()设置倒计时时间
	 start()启动倒计时
         setOnCountDownFinishListener 倒计时结束监听
